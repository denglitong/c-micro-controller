# C 语言单片机学习

## Develop toolchains

- OS: macOS Monterey 12.5.1


- Editor: Visual Studio Code
- Embedded Plugin:  Python, PlatformIO IDE
- C Compiler for Intel 8051: brew install sdcc 


- Editor: CLion
- Plugin: PlatformIO for CLion
- Platform CLI: brew install platformio
