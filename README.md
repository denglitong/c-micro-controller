# C 语言单片机学习

学习路线： 单片机 -> ARM 嵌入式系统 -> FPGA 设计 -> 上位机软件

模拟电路、数字电路、原理图设计、PCB 布板

## 开发环境

- Editor: CLion
- Plugin: PlatformIO for CLion, Ini
- Platform CLI: brew install platformio

Open the PlatformIO Home in Chrome:

```bash
pio -c clion home
```
